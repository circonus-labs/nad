# NAD MySQL

Metrics from a **local** MySQL server.


## default metrics

* Aborted_clients
* Aborted_connects
* Bytes_received
* Bytes_sent
* Connections
* Open_files
* Qcache_hits
* Qcache_inserts
* Queries
* Slow_queries
* Table_locks_waited
* Threads_connected
* Threads_running
* Uptime_since_flush_status
